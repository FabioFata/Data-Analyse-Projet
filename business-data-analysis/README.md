# Business Data Analysis — Power BI Dashboard

Ce projet consiste en une analyse de données d’entreprise réalisée à l’aide de Power BI.  
L’objectif est de transformer des données brutes en un outil décisionnel interactif permettant d’analyser la performance globale de l’entreprise et de faciliter la prise de décision.

---

## Objectifs du projet

- Exploiter des données issues d’une activité réelle d’entreprise  
- Nettoyer, structurer et modéliser les données  
- Construire des indicateurs de performance (KPI) pertinents  
- Analyser les tendances et identifier des insights clés  
- Concevoir un dashboard interactif et intuitif  

---

## Données

Les données utilisées proviennent de fichiers (CSV / Excel / base de données) représentant l’activité de l’entreprise.

Elles incluent notamment :
- Données commerciales (ventes, chiffre d’affaires, produits)  
- Données clients (segments, comportements)  
- Données temporelles (dates, périodes, saisonnalité)  
- Données opérationnelles (si applicable)  

### Préparation des données

Les données ont été :
- nettoyées (valeurs manquantes, doublons)  
- transformées (formats, colonnes calculées)  
- modélisées (relations entre tables)  

---

## Méthodologie

### Data Preparation (Power Query)

- Import des données  
- Nettoyage et transformation  
- Structuration en modèle exploitable  

### Data Modeling

- Création des relations entre tables  
- Construction d’un modèle en étoile (si applicable)  
- Création de mesures avec DAX  

### Analyse

- Calcul des KPI clés :  
  - Chiffre d’affaires  
  - Volume de ventes  
  - Performance par segment  
  - Évolution temporelle  

- Identification des tendances  
- Analyse comparative (périodes, segments, produits)  

### Visualisation

Création d’un dashboard interactif permettant :
- une vue globale de la performance  
- une analyse détaillée par dimension (temps, produit, client)  
- des filtres dynamiques (slicers)  
- une navigation fluide entre les pages  

---

## Dashboard

Le rapport Power BI est structuré en plusieurs pages :

- **Overview** : indicateurs clés et vision globale  
- **Sales Analysis** : analyse détaillée des ventes  
- **Customers** : segmentation et comportement client  
- **Products / Services** : performance par produit  
- **Time Analysis** : évolution dans le temps  
- **Insights** : synthèse des observations principales  

---

## Outils utilisés

- Power BI (Power Query + DAX + Visualisation)  
- Excel / CSV (sources de données)  
- DAX pour les mesures et indicateurs  
- Modélisation de données (schéma relationnel)  

---

## Démo

Le projet est disponible sous forme de fichier Power BI (.pbix).  
Il peut être ouvert avec Power BI Desktop pour explorer l’ensemble des visualisations interactives.

---

## Enseignements

Ce projet m’a permis de :
- maîtriser le processus complet d’analyse de données (ETL → analyse → visualisation)  
- construire un modèle de données performant  
- créer des KPI pertinents pour le pilotage d’activité  
- développer des dashboards interactifs orientés décision  
- utiliser DAX pour produire des analyses avancées
